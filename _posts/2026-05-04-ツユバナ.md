# ツユバナ（Solanum jasminoides）の活用ガイド

ツユバナ（露花）という名前は、ツルハナナスの別名や俗称として使われることが多く、朝露に濡れた姿が格別に美しいことから親しまれています。涼しげな白い花が、夏の強い日差しにさらされても次々と咲く様子は、庭に**「潤い」**をもたらしてくれます。非常に丈夫で成長が早いため、初心者でも手軽に**「壁面緑化」**に挑戦できる頼もしい植物です。

## ■ 植物プロフィール
| 項目 | 内容 |
| :--- | :--- |
| **学名 / 科名** | Solanum jasminoides / ナス科ナス属 |
| **分類** | 常緑（温暖地）〜落葉つる性低木 |
| **開花期** | 6月〜10月 |
| **花色** | 白、淡紫色 |
| **耐寒性・耐暑性** | 普通 / 非常に強い |

## ■ カテゴリー別・育て方の詳細

### 1. 栽培環境（日向・耐暑性）
日当たりの良い**「日向」**で育てると、花数が飛躍的に増えます。**「耐暑性」**に優れ、日本の多湿な夏でも元気に蔓を伸ばします。

### 2. 栽培スタイル（寄せ植え・壁面緑化）
* **寄せ植え**: 蔓を垂らす性質を活かし、大きなコンテナの背景やハンギングバスケットの主役として活躍します。
* **ローメンテナンス**: 病害虫がつきにくく、水やり以外は放任でも育つため、手軽に緑を楽しみたい方に最適です。

### 3. 水やりと肥料のタイミング
* **水やり**: 土が乾いたらたっぷりと与えましょう。夏場は特に水切れさせないようにします。
* **肥料**: 春から秋にかけて緩効性肥料を少量与えると、勢いよく成長し、花が途切れません。

---

## ■ 咲くナビ・プロの知恵：夏の清涼剤として
* **デザインのヒント**: ツユバナの白は、他の派手な夏の花（サフィニアやペンタスなど）を引き立てる名脇役。あえて同系色の白や青の花と組み合わせて、涼しげな「クールガーデン」を演出するのがプロのテクニックです。
* **冬越し**: 関東以西であれば屋外で越冬可能。落葉しても、春になれば再び勢いよく芽吹きます。

---

## ■ 花言葉とメッセージ
* **花言葉**: 「微笑み」「大きな恵み」
* **由来**: 朝の光を浴びて咲く姿が、明るい微笑みのように見えることに由来します。
