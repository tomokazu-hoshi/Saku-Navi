# アマ（Linum / 亜麻）の詳細解説

アマ（亜麻）は、リネン素材の原料として知られる歴史ある植物です。星様が設計されたカテゴリーの中でも、透き通るような「青」の花が風に揺れる姿は格別で、ナチュラルな「洋風庭園」や「ハーブ」ガーデンに欠かせない存在です。

## ■ 植物プロフィール
| 項目 | 内容 |

| **学名 / 科名** | Linum usitatissimum / アマ科アマ属 |
| **分類** | 一年草（春まき・秋まき）または宿根草 |
| **開花期** | 4月〜6月 |
| **花色** | 青、白 |
| **耐寒性・耐暑性** | 強い / 弱い（高温多湿に弱い） |

## ■ カテゴリー別・育て方の詳細

### 1. 栽培環境（日向・耐寒性）
日当たりの良い**「日向」**と、水はけの良い土壌を好みます。**「耐寒性」**は非常に強く、冬の寒さに当たることで丈夫な株に育ちます。ただし、日本の梅雨から夏にかけての高温多湿は苦手なため、風通しを良くすることが重要です。

### 2. 栽培スタイル（ハーブ・洋風庭園）
* **ハーブ**: 種子は「アマニ（亜麻仁）」と呼ばれ、健康オイルの原料となる実用的な**「ハーブ」**です。
* **洋風庭園**: 茎が細く、群生させて咲かせると幻想的な風景を作るため、イングリッシュガーデンなどの**「洋風庭園」**に最適です。

### 3. 水やりと肥料のタイミング（初心者向け）
* **水やり**: 土の表面が乾いたらたっぷりと与えます。やや乾燥気味を好むため、常に湿っている状態は避けます。
* **肥料**: それほど多くを必要としません。植え付け時に緩効性肥料を混ぜるだけで、あとは自然に育つ**「初心者向け」**な植物です。

## ■ 咲くナビ・プロの知恵：朝一番の美しさ
* **朝咲き**: アマの花は朝に開き、午後には散ってしまう「一日花」です。しかし、次から次へと新しい花が咲くため、長期間楽しめます。
* **切り花**: 蕾の状態の枝を**「切り花」**にすると、水の中でも次々と開花し、室内を爽やかに彩ります。
