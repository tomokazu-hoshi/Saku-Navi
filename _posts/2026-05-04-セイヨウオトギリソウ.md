# セイヨウオトギリソウの詳細解説

セイヨウオトギリソウ（セントジョーンズワート）は、古くからヨーロッパで魔法や魔除けの力があると信じられてきた**「ハーブ」**です。夏に咲く黄金色の花は非常に美しく、サプリメントの原料としても有名な**「有用植物」**です。

## ■ 植物プロフィール
| 項目 | 内容 |
| :--- | :--- |
| **学名 / 科名** | Hypericum perforatum / オトギリソウ科オトギリソウ属 |
| **分類** | 多年草 |
| **観賞期** | 6月〜8月 |
| **耐寒性・耐暑性** | 強い / 強い |

---

## ■ 花言葉とメッセージ
* **花言葉**: 「迷信」「敵意」「恨み」
* **由来**: 魔除けに使われた歴史や、葉にある黒い斑点に由来します。
