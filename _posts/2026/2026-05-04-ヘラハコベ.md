# ヘラハコベ（Stellaria saxatilis）の詳細解説

ヘラハコベは、春の野山や日当たりの良い道端で見かける、清楚な白い花が特徴の**「一年草」**です。通常のハコベよりも葉が細長い「ヘラ型」をしていることからその名がつきました。非常に**「ローメンテナンス」**で、日本の自然な風景によく溶け込む**「野草」**としての魅力を持っています。

## ■ 植物プロフィール
| 項目 | 内容 |
| :--- | :--- |
| **学名 / 科名** | Stellaria saxatilis / ナデシコ科ハコベ属 |
| **分類** | 一年草（または越年草） |
| **開花期** | 4月〜6月 |
| **花色** | 白 |
| **耐寒性・耐暑性** | 強い / 弱い（夏前に一生を終える） |

## ■ カテゴリー別・育て方の詳細

### 1. 栽培環境（日向・耐寒性）
日当たりの良い**「日向」**を好みます。寒さの中で芽を出し成長するため**「耐寒性」**があり、春の訪れとともに一気に開花します。

### 2. 栽培スタイル（ローメンテナンス・野草）
* **ローメンテナンス**: 自生種のため非常に強く、水やりや肥料はほとんど必要ありません。
* **野草**: 密集して咲く姿は、白い小花が散りばめられたような素朴な美しさがあります。

### 3. 水やりと肥料のタイミング
* **管理**: 地植えであれば放任で問題ありません。肥料は不要ですが、土壌が豊かであればより大株になります。

---

## ■ 咲くナビ・プロの知恵：星のような花
* **学名の由来**: 属名の「Stellaria」はラテン語で「星」を意味します。花びらが深く2つに割れているため、5枚の花びらが10枚あるように見え、まさに小さな白い星が散らばっているような愛らしさです。

---

## ■ 花言葉とメッセージ
* **花言葉**: 「愛のぬくもり」「密会」
* **由来**: 小さな花が集まって寄り添うように咲く様子にちなんでいます。
